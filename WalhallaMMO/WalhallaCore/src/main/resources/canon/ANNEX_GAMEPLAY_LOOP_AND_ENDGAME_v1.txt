# GAMEPLAY LOOP AND ENDGAME PROGRESSION

## EARLY GAME (LEVELS 1-10)
PHASE:Early Game
FOCUS:Learning mechanics
CONTENT:Tutorial quests, basic combat
REWARDS:Starting equipment, initial currency
DURATION_HOURS:3-5

## MID GAME (LEVELS 11-30)
PHASE:Mid Game
FOCUS:Branch specialization and professions
CONTENT:Dungeon instances, profession leveling, guild formation
REWARDS:Uncommon gear, profession unlocks
DURATION_HOURS:15-25

## LATE GAME (LEVELS 31-45)
PHASE:Late Game
FOCUS:Endgame raids and legendary gear
CONTENT:5-player dungeons, guild wars, world bosses
REWARDS:Epic and legendary gear
DURATION_HOURS:40-60

## ENDGAME (LEVEL 45+)
PHASE:Endgame
FOCUS:Guild competition and prestige
CONTENT:Endless dungeons, PvP arenas, speedruns
REWARDS:Cosmetics, titles, leaderboard positions
DURATION_HOURS:Unlimited
