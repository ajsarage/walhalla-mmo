STEP 3: Start Server

   > /plugins

STEP 4: Verify New Features
------------------------------------
Test each new system:

1. Daily Rewards:
   > /daily
   Should show "You claimed X WCoin and Y WCraft!"

   - Player 2: /party accept Player1
   - Both: /party list (should show 2 members)
   - Test: /pc Hello party!

3. Skill System:
   > /skills
   Should open GUI menu with talent nodes

4. Action Bars:
   - Gain XP (kill mobs, complete quests)
   - Should see action bar message: "+X XP"

ğŸ› TROUBLESHOOTING
====================================

Problem: Plugin won't load
Solution: 
- Check Java version: java -version (needs 21+)
- Check Paper version: version command (needs 1.21+)
- Check console for error messages

Problem: /daily command not found
Solution:
- Verify WalhallaCore loaded successfully
- Check plugins/WalhallaCore/plugin.yml has daily command
- Reload with: /reload confirm

Problem: /party or /skills not working
Solution:
- Verify WalhallaParty and WalhallaSkills loaded
- Check /plugins shows them in green
- Verify JARs copied correctly

Problem: "Cannot find symbol" errors
Solution:
- This is a compile error, not runtime
- Re-run: mvn clean install -DskipTests
- Verify all 18 JARs exist in target/ folders
