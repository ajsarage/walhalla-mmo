================================================================================
TROUBLESHOOTING
================================================================================

Build fails with "Cannot find symbol"
  ‚Üí Solution: mvn clean

Paper API version error
  ‚Üí Check: pom.xml paper.api.version property
  ‚Üí Update if needed: <paper.api.version>1.21.8-R0.1-SNAPSHOT</paper.api.version>

Plugin won't load on server
  ‚Üí Check: Server is Paper 1.21+
  ‚Üí Check: .jar file is in plugins/ folder
  ‚Üí Check: Server logs for error messages
  ‚Üí Check: plugin.yml is valid YAML

Economy not working
  ‚Üí Check: CoreEconomyService registered properly
  ‚Üí Check: Player data folder has write permissions
  ‚Üí Check: economy_ledger.log exists
