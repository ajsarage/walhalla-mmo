OBJETIVO
--------
Definir de forma EXPLÍCITA y CANÓNICA el catálogo completo
de ELEMENTOS COMBINADOS del proyecto WalhallaMMOonline.

Este documento fija:
- Qué pares de elementos base existen
- Qué nombre canónico recibe cada combinación
- Qué combinaciones están creadas en código
- Qué combinaciones están planificadas pero no creadas

Este anexo es la ÚNICA fuente de verdad
para combinaciones elementales.

====================================================================
REGLA GENERAL DE COMBINACIÓN
====================================================================

- Un elemento combinado es un PAR NO ORDENADO de elementos base
- Cada par tiene UN ÚNICO nombre canónico
- El nombre canónico define la identidad del combinado
- La existencia en CANON NO implica existencia en código
- Ningún combinado crea un nuevo elemento base

REGLAS DE USO
====================================================================

1. Ningún elemento combinado es utilizable en FASE 6
2. Ningún bastón combinado existe en FASE 6
3. Los spells combinados existentes en código:
   - permanecen BLOQUEADOS
   - se clasifican como SPELL_BLOQUEADO o SPELL_REUTILIZABLE
4. La habilitación de combinados requiere:
   - contrato específico
   - phase gating explícito
   - decisión de diseño posterior
