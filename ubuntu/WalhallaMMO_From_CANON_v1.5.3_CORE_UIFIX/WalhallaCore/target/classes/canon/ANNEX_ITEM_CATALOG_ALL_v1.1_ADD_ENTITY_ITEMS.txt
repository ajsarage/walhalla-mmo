================================================
1. PROPÓSITO
================================================
Este addendum amplía ANNEX_ITEM_CATALOG_ALL_v1 para incluir
TODOS los ítems necesarios que pueden obtenerse por ENTIDADES
(mobs) en Minecraft 1.21.8 y que impactan crafting/economía.

================================================
2. REGLAS
================================================
1) Estos ítems pasan a existir canónicamente y pueden ser usados
   por refining/crafting/economía.
2) Calidad: FIXED_Q1 (drops) salvo piezas “equipables” (si existieran),
   que se definirían en anexos de equipo.
3) Stack:
   - 64 para recursos comunes
   - 16 para recursos potentes/raros
   - 1 para objetos únicos de boss

================================================
3. CAMPOS (FIJOS POR DEFECTO)
================================================
Tipo: RESOURCE (salvo que se indique MATERIAL/COMPONENT)
Tier: T1–T8 según tabla de este addendum
Calidad: FIXED_Q1
Uso: PVE_LOOT_INPUT, CRAFTING_INPUT, ECON_TRADE
Stack: según especificación individual
