================================
1. PROPÓSITO
================================
Este documento define el catálogo COMPLETO y CERRADO de ítems existentes
en WalhallaMMOonline.

Ningún sistema del juego (gathering, crafting, economía, combate, progresión,
endgame) puede utilizar ítems que no estén definidos explícitamente en este
anexo o en una versión posterior del mismo.

Este anexo NO define mecánicas, solo DATOS.

================================
2. RELACIÓN CON OTROS DOCUMENTOS
================================
Este anexo se apoya y no contradice:

- WALHALLA_CANON_UNIFICADO_FINAL_v4
- CANON_ITEM_TIERS_AND_QUALITY_v1
- CANON_PROFESSIONS_MASTER_v1
- CANON_CRAFTING_SYSTEM_v2

- CANON_COMBAT_MASTER_v1

================================
3. REGLAS MAESTRAS DEL CATÁLOGO
================================
1) Todo ítem tiene un ID canónico único.
2) No existen ítems implícitos.
3) No existen variantes fuera de calidad (Q0–Q5).
4) Todo ítem pertenece a una sola categoría primaria.
5) El Tier es obligatorio salvo excepciones explícitas.
6) La calidad nunca crea nuevos IDs.
7) Todo ítem debe poder ser referenciado por otros anexos.

================================
4. CONVENCIÓN DE IDS
================================
Formato:
IT_<CATEGORIA>_<FAMILIA>_<NOMBRE>[_T#]

Reglas:
- Mayúsculas
- Sin espacios
- Sin caracteres especiales
- El tier solo se incluye si el ítem es intrínsecamente de tier fijo
