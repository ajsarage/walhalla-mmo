===========================
3. REGLAS GENERALES DE EQUIPO
===========================
1) Todo equipo pertenece a un tier T1–T8.
2) El tier define:
   - poder base
   - dificultad de obtención
   - zona de uso recomendada
3) El equipo NO escala de tier.
4) La calidad (Q0–Q5) modifica eficiencia,
   NO el tier.
5) El equipo se craftea o se obtiene por rutas
   canónicas (nunca implícitas).
6) El equipo tiene durabilidad (canon global).
7) Ningún equipo existe fuera de este sistema.

===========================
4. DEFINICIÓN DE TIERS
===========================

--------------------------------
T1 — INICIAL
--------------------------------
Rol:
- Inicio de progresión
- Zonas seguras

Equipo:
- Madera básica
- Piedra simple
- Tela ligera

Obtención:
- Crafteo básico
- Vanilla integrado

--------------------------------
T2 — APRENDIZ
--------------------------------
Rol:
- Progresión temprana
- Primer combate real

Equipo:
- Metal básico
- Tela tratada
- Componentes simples

Obtención:
- Crafteo estructurado
- Dependencia mínima de oficios

--------------------------------
T3 — EXPERIMENTADO
--------------------------------
Rol:
- Juego medio bajo
- Builds iniciales

Equipo:
- Metal trabajado
- Armaduras completas
- Armas definidas

Obtención:
- Crafteo multi-oficio
- Uso activo del herrero

--------------------------------
T4 — VETERANO
--------------------------------
Rol:
- Juego medio
- Especialización de rol

Equipo:
- Aleaciones
- Componentes avanzados
- Bastones mágicos estables

Obtención:
- Crafteo avanzado
- Drops de entidades relevantes

--------------------------------
T5 — ÉLITE
--------------------------------
Rol:
- Transición a endgame
- Riesgo elevado

Equipo:
- Materiales raros
- Componentes mágicos
- Joyería funcional

Obtención:
- Zonas peligrosas
- Dependencia clara de joyería y magia

--------------------------------
T6 — MAESTRO
--------------------------------
Rol:
- Endgame temprano
- Builds cerradas

Equipo:
- Materiales exóticos
- Artefactos parciales
- Equipos híbridos

Obtención:
- Drops élite
- Crafteo complejo
- Costes elevados

--------------------------------
T7 — LEGENDARIO
--------------------------------
Rol:
- Endgame avanzado
- Contenido de alto riesgo

Equipo:
- Componentes extremos
- Artefactos mayores
- Sinergias completas

Obtención:
- Bosses
- Zonas de máximo peligro
- Economía fuerte

--------------------------------
T8 — MÍTICO
--------------------------------
Rol:
- Endgame absoluto
- Meta del juego

Equipo:
- Materiales primordiales
- Artefactos definitivos
- Sets completos

Obtención:
- Boss final
- Crafteo extremo
- Uso de WCraft

5. CLASES DE EQUIPO
===========================

--------------------------------

- Lanzas
- Dagas
- Arcos
- Ballestas

Tiers: T1–T8

--------------------------------
5.2 ARMADURAS
--------------------------------
Oficio: HERRERO
Soporte: SASTRE

Tipos:
- Ligera
- Media
- Pesada

Piezas:
- Casco
- Pechera
- Piernas
- Botas

Tiers: T1–T8

--------------------------------
5.3 BASTONES MÁGICOS
--------------------------------
Oficio: DON_DE_HIERBAS

Elementos:
- Fuego
- Agua
- Aire
- Tierra
- Luz
- Oscuridad

Tiers: T1–T8

--------------------------------
5.4 ESCUDOS
--------------------------------
Oficio: HERRERO

Rol:
- Defensa
- Soporte de builds tanque

Tiers: T1–T8

--------------------------------
5.5 EQUIPO ESPECIAL
--------------------------------
Oficios:
- DON_DE_HIERBAS (artefactos)
- TRAMPERO / INGENIERO (dispositivos)

Incluye:
- Artefactos mágicos
- Trampas
- Dispositivos ofensivos

Tiers: T4–T8

===========================
6. REQUISITOS DE USO
===========================

Para usar equipo de un tier T# se requiere:

- Nivel global adecuado
- Progresión de rama correspondiente
- Afinidad mínima (si aplica)
- Acceso a la zona del tier

El equipo NO se bloquea por oficio,
solo por progresión del personaje.

===========================
7. RELACIÓN CON PROGRESIÓN
===========================

- El equipo es el principal vector de poder.
- La progresión de equipo impulsa:
  - acceso a zonas
  - acceso a bosses
  - endgame activities

No existe progresión de poder
fuera del sistema de tiers.

===========================
8. EXPANSIÓN FUTURA
===========================

Nuevos tipos de equipo:
- deben asignarse a un tier existente
- deben respetar esta jerarquía
- se añaden vía NOXO o nuevo anexo
