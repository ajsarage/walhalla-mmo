============================
1. PROPÓSITO
============================
Este documento define TODAS las recetas de REFINADO
existentes en WalhallaMMOonline.

El refinado es la transformación de:
- recursos brutos
- drops de entidades
en materiales utilizables para crafting.

Este documento NO define recetas finales de equipo,
solo procesos intermedios.

============================
3. REGLAS GENERALES DE REFINADO
============================
1) Toda receta usa UNA estación concreta.
2) Toda receta tiene inputs y outputs explícitos.
3) El tier del output es IGUAL al tier del input.
4) El refinado NO genera calidad (Q siempre FIXED_Q1).
5) El refinado puede consumir moneda (sink).
6) No existen recetas implícitas.

============================
4. DEFINICIÓN DE CAMPOS
============================
Estación:
ID definido en ANNEX_CRAFTING_STATIONS_v1

Input:
ID exacto del catálogo + cantidad

Output:
ID exacto del catálogo + cantidad

Tiempo:
Tiempo base de proceso (segundos)

Coste:
WCoin consumido por ejecución
